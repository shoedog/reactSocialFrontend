Welcome.

Windows OS Environment
---
- Setup:  ```npm install```
- Build: ```npm run win-build```
- Run:  ```npm run win-dev```

Webpack-dev-server (localhost:8080) & node server (localhost:3000)

Mac OS Environment
---
- Setup:  ```npm install```
- Build: ```npm run build```
- Run:  ```npm run dev```

Webpack-dev-server (localhost:8080) & node server (localhost:3000)

Usage
---

Start the development server with this command:

```
npm start
```
